# Reports Directory

이 폴더는 선수용 공유 리포트 JSON 파일들이 저장되는 곳입니다.

## 자동 업로드
리포트 페이지의 "🔗 선수용 링크 생성" 버튼을 클릭하면 GitHub API를 통해 자동으로 이 폴더에 파일이 추가됩니다.

## 파일명 규칙
`YYYYMMDD-선수이름-XXXX.json` (XXXX는 4자리 랜덤 문자)

## 접근 URL
파일이 추가되면 다음 URL로 자동 접근 가능:
```
https://kkl0511.github.io/Uplift-Labs-Report/#/r/<파일명_확장자제외>
```

## 정리
오래된 리포트는 수동으로 삭제하세요 (이 폴더에서 파일 삭제 → push).
